# iOS group chat denial of service

### SUMMARY BY LY CORPORATION

If an invalid E2EGgroupKey is shared within the group and a message encrypted with it is received, LINE iOS will crash, making it difficult to run the client. CVE-2022-41568 has been issued for this vulnerability.
