# Web API key registration allows registering multiple keys by reusing `request_id` 

### SUMMARY BY VALVE

The Steam API key registration now requires a confirmation via the mobile authenticator. The request_id, once correctly confirmed, was not correctly updated and could be used more than once to register keys for the given account.

The registration process is now fixed, and accounts with multiple registered keys have been corrected.
