# Using Branded Hashtag Feature User Partnered with Account Manager Can View Videos Uploaded By A Private TikTok Account If 'item_id' Is Known

### SUMMARY BY TIKTOK

The 'branded hashtag challenge' feature on TikTok could have allowed anyone that had been partnered with an account manager to request a TikTok video ID and view the accompanying video. If the ID of a private TikTok video was known and sent, the user would have been shown the private video of another creator. This has been fully remediated. We thank @dxcoder for reporting this to our team.
