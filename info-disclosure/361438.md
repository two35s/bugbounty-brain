# Open AWS S3 bucket at ubergreece.s3.amazonaws.com exposes confidential internal documents and files

### SUMMARY BY UBER

The Uber Greece AWS S3 bucket was open, allowing any remote user to view and download the files. Some of these files included confidential internal documents which could negatively impact Uber's brand.

### RESEARCHER SUMMARY

I found a uber microsite that had a link to this AWS bucket being used by Uber greece. Had some internal files and it was fixed quickly.
