# Information Disclosure PHPpgAdmin

### SUMMARY BY RATELIMITED

PHPpgAdmin is a piece of script which allows system administrators to manage their Postgres databases easily from a webUI.

We had forgotten to limit access to this script, resulting in the ability for a brute-force attack to happen.
