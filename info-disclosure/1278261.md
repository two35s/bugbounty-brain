# Uninstalling Rockstar Games Launcher for Windows (64-bit), then reinstalling keeps you logged in without authentication

### SUMMARY BY ROCKSTAR GAMES

In this report, the researcher identified a potential weakness in Rockstar Games Launcher that caused the application to retain profile data on the local machine, even after the application was uninstalled. This included auto sign-in flags, resulting in automatic sign-ins when reinstalling Rockstar Games Launcher. To address this, an option was added to the application's Uninstaller to remove all stored profile data from the device.
