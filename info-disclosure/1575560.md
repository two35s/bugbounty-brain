# Internal Employee informations Disclosure via TikTok Athena api

### SUMMARY BY TIKTOK

A vulnerability was found where internal employee email address could be disclosed on a TikTok Ads endpoint API by adding an "employee_email" value to the "metrics" field/array in the request body. We thank @hein_thant for reporting this to our team.
