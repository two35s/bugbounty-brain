# Flickr API key leaked in GitHub commit

### SUMMARY BY MOZILLA

A security researcher discovered a leaked Flickr API key in a public GitHub repository belonging to Mozilla. The key is used to pull public data from Flickr and did not pose a security risk.

This report is out of scope of our program and was resolved since we were able to revoke the API key. It is not eligible for bounty. Similar reports would be closed as informative.
