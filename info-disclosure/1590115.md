# Production Key and Data Found on Subdomain No Longer Operated by Shopify / Dangling DNS

### SUMMARY BY SHOPIFY

The disused subdomain for an app's data center had a dangling DNS record, and it was pointing to an IP address that doesn’t belong to Shopify. We removed the DNS record, but no Shopify services were impacted by the presence of this dangling DNS record, so the report was ineligible for a bounty.
