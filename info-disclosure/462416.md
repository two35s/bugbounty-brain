# Grammarly Keyboard for Android <4.1  leaks user input through logs (except for sensitive input fields)

### SUMMARY BY SUPERHUMAN (FORMERLY GRAMMARLY)

Currently, Grammarly for Android doesn't support devices with platform versions less than Android 5.0.
