# CSRF in Delete Pet Function

### SUMMARY BY MARS

A Cross-Site Request Forgery (CSRF) vulnerability is identified in the Delete Pet functionality on the ████████ platform. The vulnerability is discovered in the pet deletion endpoint which can be exploited by manipulating the pet ID parameter. When exploited, an attacker can force authenticated users to delete their pets without their knowledge or consent by simply having them visit a maliciously crafted webpage containing the CSRF payload.
