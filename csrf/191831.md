# Cross-site request forgery (CSRF) vulnerability on a DoD website

### SUMMARY BY U.S. DEPT OF DEFENSE

A cross-site request forgery vulnerability was found on a Department of Defense website which could trick a web user into posting or changing confidential information, or redirecting sensitive information to the malicious user. @korprit was able to demonstrate this vulnerability by crafting a specially formatted webpage.
