# Уязвимость Создание фотографий без ведома пользователей

### SUMMARY BY VK.COM

CSRF, позволяющий загрузить фотографию на сайт от имени атакуемого, без публикации.

### RESEARCHER SUMMARY

При переходе на специально сгенерированную ссылку VK.COM 
(или на сайт с вредоносным кодом) на ID пользователя автоматически добавлялась фотография (моя)
