# Client-Side Path Traversal on LINE Developers Console

### SUMMARY BY LY CORPORATION

LINE Developers Console (https://developers.line.biz) had a Client-Side Path Traversal (CSPT) vulnerability that led to an effective CSRF. Although the operations that could be enforced with the CSRF were limited, some availability could be compromised.
