# subdomain Takeover at blog.exchangemarketplace.com 

### SUMMARY BY SHOPIFY

@m7mdharoun noticed that blog.exchangemarketplace.com had been configured with a DNS record pointing at Shopify, but that the domain had not been claimed. This allowed him to add the domain to his own store. Because the domain had not been used (and plans to use it had been abandoned) the security impact of this subdomain takeover was minimal.
