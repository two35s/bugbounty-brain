# Cloudflare /cdn-cgi/ path allows resizing images from unauthorised sources on enjinusercontent.com

### SUMMARY BY ENJIN

@19whoami19 identified that one of Enjin's websites (`enjinusercontent.com`) had a misconfiguration within Cloudflare, due to being on a legacy pricing model, which enabled resizing images outside of the domain for which the request was initiated. A toggle was introduced for users of the legacy pricing model which allowed disabling this functionality without direct engagement with Cloudflare. The functionality was disabled, cache was purged, and the issue confirmed as having been resolved.
