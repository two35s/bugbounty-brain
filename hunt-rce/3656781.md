# PHP code injection via delivery limitation logical 

### SUMMARY BY REVIVE ADSERVER

HackerOne community member 0x4c616e has reported a missing validation of user input when saving delivery limitations in Revive Adserver 6.0.6 and earlier. A low‑privileged user could use the `logical` parameter to inject malicious PHP code into the `compiledlimitations` field on the database and have it executed during banner delivery.

Input sanitisation has been improved to ensure that the parameter is properly validated.
