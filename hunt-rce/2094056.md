# RCE and DoS in Cosmovisor

### SUMMARY BY COSMOS

A bug in early versions of cosmovisor that could lead to DOS or potential RCE depending on the host configuration was discovered. 

### RESEARCHER SUMMARY

Really bad vulnerability in old version of cosmos-sdk can lead to RCE at worst, DoS at best, and could still be active even in otherwise updated nodes on cosmos chains. 

We would like to get a security advisory out there in order to ensure that users are not running a vulnerable version of the Cosmovisor. We found a print sink that works on all chains we've tested to trigger this issue. We felt this was the best means of communicating with the developing in a private channel about the exploitability of this bug.

For more on the vulnerability, read about the bug at https://maxwelldulin.com/BlogPost/stdout-cosmos-sdk-rce.
