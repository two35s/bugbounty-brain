# Time Based SQL Injection vulnerability on a DoD website

### SUMMARY BY U.S. DEPT OF DEFENSE

A Department of Defense webserver was vulnerable to a SQL injection attack that could have revealed sensitive financial information. @psychomantis was able to demonstrate this vulnerability by crafting specially formatted URLs. Thank you!


### RESEARCHER SUMMARY

The service being used didn't properly sanitize id parameter. This resulted in a Blind SQL Injection vulnerability and it was demonstrated by making the SQL server wait for a period of time before responding.
