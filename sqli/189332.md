# SQL injection vulnerability on a DoD website

### SUMMARY BY U.S. DEPT OF DEFENSE

A Department of Defense website was vulnerable to a SQL injection attack which may allow an attacker to execute arbitrary SQL commands and expose sensitive data. @mthirup was able to demonstrate this vulnerability by crafting a specially formatted URL. Thanks to @mthirup for discovering this vulnerability!
