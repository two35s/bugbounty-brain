# Multiple Open Redirect on TikTok domains

### SUMMARY BY TIKTOK

An open redirection vulnerability was found via a path traversal on the 'redirect_url' parameter when logging in on a TikTok Seller domain, which could have resulted in takeover of a TikTok Seller account. We thank @ashrafabdelrazik for reporting this to our team.
