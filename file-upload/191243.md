# File upload vulnerability on a DoD website

### SUMMARY BY U.S. DEPT OF DEFENSE

A Department of Defense website was misconfigured in a manner that may have allowed an attacker to bypass tests for malicious files when uploading content. @korprit was able to demonstrate this vulnerability by altering the file sufficiently to defeat signature based analysis. Thanks @korprit!
