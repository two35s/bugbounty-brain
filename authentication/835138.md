# [www.drive2.ru] Insufficient Security Configurability - Email notification is not being sent while changing passwords

### SUMMARY BY DRIVE.NET, INC.

Email notification was not sent while changing passwords. This issue was fixed.

### RESEARCHER SUMMARY

Best Practices As recommended practices, Due to missing notification email when changing password, If the password has been maliciously changed, the user will not be able to notice it, so immediate security measures cannot be taken.
