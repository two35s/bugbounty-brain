# OAUTH pemission set as true= lead to authorize malicious application

### SUMMARY BY COINBASE

OAuth authorize button in the Coinbase Android App did not have the android:filterTouchesWhenObscured attribute set to true, which may have made it vulnerable to tap-jacking.


### RESEARCHER SUMMARY

Reported `UI Redressing (Clickjacking)` For ___Mobile application of Coinbase___ , attack scenario is same as www.paulosyibelo.com/2015/06/no-xfo-youre-d-silly-bugs-with-real-harm.html which is reported by @paulos_
