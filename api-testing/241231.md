# flash injection in http://www.rockstargames.com/IV/imgPlayer/imageEmbed.swf

### SUMMARY BY ROCKSTAR GAMES

In this report, the researcher found a vulnerability in one of our Flash players (http://www.rockstargames.com/IV/imgPlayer/imageEmbed.swf) that allowed loading of external .swf files. This in turn allowed the researcher to perform a XSS attack.
