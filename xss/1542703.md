# Stored XSS on TikTok Live Form

### SUMMARY BY TIKTOK

A Cross-Site Scripting (XSS) vulnerability was found on a restricted endpoint by an authenticated user inputting a payload in the "Description" form when creating a LIVE event on the TikTok mobile app. We thank @aidilarf_2000 for reporting this to our team.
