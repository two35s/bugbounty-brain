# Stored xss in larksuite internal helpdesk and other user's helpdesk.

### SUMMARY BY LARK TECHNOLOGIES

A stored XSS (cross site scripting) vulnerability was found which an attacker could have potentially used to obtain access to the internal team's help desk and view submitted user tickets. We have resolved this issue and thank @imran_nisar for reporting this to our team.
