# IE search XSS

### SUMMARY BY GENERAL MOTORS

Gm.com website's search function contained a parameter that allowed XSS injection. The vulnerable input parameter has been identified and remediated.

### RESEARCHER SUMMARY

Search DOM XSS reproduced in IE due to non-encoded GET params in this browser.
The <script xxx> tag with an extra parameter and without the end tag used to bypass the XSS filter and some JS code restrictions.
