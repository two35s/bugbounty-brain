# DOM Based XSS on a DoD website

### SUMMARY BY U.S. DEPT OF DEFENSE

A DoD website was vulnerable to a DOM based cross-site scripting attack which may be used to trick a web user into executing a malicious script, potentially revealing a user's browser cookies or modify web content. @juliocesar was able to demonstrate this vulnerability by crafting a specially formatted URL.
