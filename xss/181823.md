# Stored XSS в личных сообщениях

### SUMMARY BY VK.COM

Lack of data filtration, which leads to stored XSS in instant messenger service.

### RESEARCHER SUMMARY

Stored XSS in personal messages. The vulnerability exists due to the lack of filtering data which is taken from the META tags for demonstration preview of the site.
