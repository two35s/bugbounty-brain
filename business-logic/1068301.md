# File sizes may be manipulated into negative numbers when uploading

### SUMMARY BY LY CORPORATION

When uploading a large file, a message indicating that there is not enough storage space is displayed. However, because the file size is recalculated and transmitted during upload, if the file is altered to have a negative size, the upload is allowed.
