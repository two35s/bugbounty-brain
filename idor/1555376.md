# IDOR on Tagged People

### SUMMARY BY TIKTOK

An Insecure Direct Object Reference (IDOR) vulnerability was found on the TikTok app where a user was able to tag any other user on videos that don't belong to them. We thank @apapedulimu for reporting this to our team.
