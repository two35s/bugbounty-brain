# China - IDOR on Reservation Staging/Non Production Site - https://reservation.stg.starbucks.com.cn

### SUMMARY BY STARBUCKS

xmfc discovered that the staging/non production site at https://reservation.stg.starbucks.com.cn/api/customer/reservation/history allowed anyone to retrieve fabricated test user reservation data by providing a OTP of 111111.
@xmfc — thank you for reporting this vulnerability and for confirming the resolution.
