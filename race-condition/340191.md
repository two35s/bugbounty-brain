# Session works after logout from Shopify account

### SUMMARY BY SHOPIFY

@Cryptographer reported if a logout request for a given session was received during the time a product creation request was in progress from the same session, it was possible the logout request could fail. We determined this was the result of a race condition in how we were updating and revoking user sessions and have since fixed the issue.
